DCRYPT is a blog about uncovering the layers of topics in design, art, music, fashion, culture and the many mysteries of life. It explores and addresses overlooked + unseen phenomenons. Visuals in this project are completely experimental, where the fusion of ancient blackletter gothic script and a modern programming font join to clash. The web applicaton consists of both frontend and backend development, created using HTML, CSS, Javascript, AngularJS, NodeJS and MongoDB. It features file uploading, custom colour styling and a search tool. Visit dcrypt.net.
